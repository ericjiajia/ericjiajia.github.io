display:table-cell 使用心得

发表于 2017-07-22 | 分类于 基础知识

display:table-cell在日常工作使用后，你就会发现这货简直可以称作布局神器，对于一些令人头疼的布局问题总能给出一些非常好的解决方案，所以在这里专门做一些总结：

两div实现等高对齐

上图中只给左侧div设置了display:table-cell，右侧则会随着左侧的高度改变而改变高度。
若想实现两个互相跟随改变，给右侧div也设置display:table-cell即可。

弹性、响应式布局

给父级元素设置display:table，子级元素设置display:table-cell，则自己元素会跟随兄弟元素的高度而改变，也会根据父级元素高度的改变而改变。
适合响应式布局。
